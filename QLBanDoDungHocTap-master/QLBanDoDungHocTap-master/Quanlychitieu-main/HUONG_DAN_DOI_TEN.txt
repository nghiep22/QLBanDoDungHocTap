HƯỚNG DẪN ĐỔI TÊN FOLDER

Bước 1: Đóng VS Code và tất cả ứng dụng đang mở folder này

Bước 2: Đổi tên các folder theo thứ tự:

1. Đổi tên folder:
   "QLBanDoDungHocTap-master\Quanlychitieu-main" 
   → "QLBanDoDungHocTap-master\QLBanDDHT"

2. Đổi tên folder gốc:
   "QLBanDoDungHocTap-master" 
   → "QLBanDDHT"

Bước 3: Cập nhật file Solution

Mở file: QLBanDDHT\be\Quanlychitieu.sln
Đổi tên thành: QLBanDDHT.sln

Bước 4: Mở lại VS Code

Mở folder: QLBanDDHT\be

Bước 5: Build lại

Mở terminal và chạy:
dotnet build QLBanDDHT.sln

HOÀN TẤT!

Cấu trúc sau khi đổi tên:
